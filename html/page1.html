Welcome to Karaoke

If you have got this far you must have successfully installed Karaoke on your computer, and must have the required support packages installed, these are.

Python to run the program
WxPython the support libraries to run everything
pillow for the round buttons
VLC for the media playback

If you don't have them, wxpython and pillow can easily be installed using pip on the command line.

pip install pillow

pip install wxpython

Note that VLC is at the time of writing a 32 bit application on windows, so the 32 bit version of Python 2.7 is needed if running from inside a Python enviroment.

Linux users can use 64 bit Python 2.7.
